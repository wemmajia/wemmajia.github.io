Computer Science Courses and Professor Reviews


COMP110: Introduction to Programming and Data Science

Instructor: Kris Jordan

Language: Python

Comp 110 is a great class to take if you're curious about the CS major; Kris Jordan is an amazing professor! No prior programming experience is needed, but you'll have to put in the effort to do well.


COMP210: Data Structures and Analysis.

Instructor: Paul Stotts

Language: Java

Comp 210 is the most important class you will take in terms of internship and job hunt preparation, so pay attention (even if lectures are boring at times). Make sure you put on your thinking caps when you learn about big O. They ask that in interviews. This class is in Java but the professor will expect you to learn the language on your own, so be prepared to do a lot of Googling.


COMP211: System Fundamentals.

Instructor: Brent Munsell

Language: C

Comp 211 takes you down to the very fundamental of how a system works, such as knowing how to write in binary and understanding assembly code. There are very specific things you learn and although it is important information, what path you decide to choose after graduating may affect whether this class would actually be relevant to your career.


COMP301: Foundations of Programming

Instructor: Aaron Smith

Language: Java

Comp 301 is an important class to take for conceptual technical interviews. There are a lot of details to remember, but Aaron is a great lecturer and his slides are super helpful.


COMP311: Computer Organizaton

Instructor: Brent Munsell

Language: MIPS

The class is similar to 211, it will be difficult. Programming in MIPS will be a pain.


Recommended Courses

COMP 126: Although you dont receive any credit for the class, it's a great way to learn HTML/CSS and prepare you for Comp 426: Web Programming. You can also get a personal project at the end of the class to spice up your resume.
